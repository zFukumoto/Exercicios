Como Usar

Simplificado

1º Expessura da Borda; 2º Estilo da Borda; 3º Cor da Borda

Exemplo:

'border: 3px solid darkorange;'

Detalhado

Assim como no Simplificado, existem os elementos da borda, sendo eles a espessura, o estilo e a cor da borda, porém no detalhado, cada elemento pode ser definido separadamente.

Os elementos são:

  • border-width
  • border-style
  • border-color
Sendo respectivamente a expessura, o estilo e a cor da borda.

Além disso, nesses elementos podemos especificar quais bordas queremos aplicar a estilização, também é possível seguir uma ordem específica para definir as bordas: Superior, Direita, Inferior e Esquerda.
Ou podemos definir cada borda separadamente, utilizando os seguintes elementos:

  • border-top
  • border-right
  • border-bottom
  • border-left

Exemplos:

  • 'border-top: 5px dashed blue;'
  • 'border-right: 2px solid red;'
  • 'border-bottom: 4px dotted green;'
  • 'border-left: 3px double purple;'

Estilos de Bordas

Os estilos de bordas são os tipos de linhas que podem ser aplicadas ao redor de elementos HTML. Cada estilo tem um aspecto diferente e pode ser usado para criar diferentes efeitos visuais.

Estilos:

Solid Dashed Dotted Double
Inset Outset Groove Ridge

Extra

Arredondar Borda: 'border-radius: 10px;'
